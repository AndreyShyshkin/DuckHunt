Гра **DuckHunt** на Python (бібліотека `pygame`)

## Запуск гри
1. Встановіть необхідні бібліотеки за допомогою pip:
   ```
   pip install -r requirements.txt
   ```
2. Запустіть файл `main.py`:
   ```
   python main.py
   ```

### Аргументи командного рядка
Ви можете налаштувати гру при запуску за допомогою наступних аргументів:

* `--width`: Встановити ширину екрану (за замовчуванням: 800)
* `--height`: Встановити висоту екрану (за замовчуванням: 500)
* `--volume`: Встановити гучність звуків (від 0.0 до 1.0, за замовчуванням: 1.0)
* `--difficulty`: Встановити рівень складності (швидкість птахів) (1: Нормально, 2: Швидко, 3: Важко, за замовчуванням: 1)

Приклад запуску з користувацькими налаштуваннями:
```bash
python main.py --width 1024 --height 768 --volume 0.5 --difficulty 2
```
